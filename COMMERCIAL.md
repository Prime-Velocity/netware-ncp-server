# Commercial Licensing & Genesis Ecosystem

The `netware-ncp-server` open source code is free to use under the
[Apache License 2.0](LICENSE).

The following use cases require a **Genesis Systems Commercial License**:

## What requires a commercial license

- **Genesis ecosystem integration** — embedding or distributing this implementation
  as part of any Genesis platform, appliance, or managed service offering
- **STCS Certification** — representing a product or service as "STCS-certified"
  or "Genesis-compatible" requires a formal certification agreement
- **OEM / white-label redistribution** — shipping this code inside a proprietary
  product under a different name or brand
- **Hosted / SaaS deployment** — offering a paid hosted service built primarily
  on this codebase

## What does NOT require a commercial license

- Internal use at any scale under Apache 2.0
- Research, academic, and non-commercial open-source projects
- Forking and publishing your own Apache 2.0 improvements
- Building products that *use the STCS protocol* but do not ship this code
